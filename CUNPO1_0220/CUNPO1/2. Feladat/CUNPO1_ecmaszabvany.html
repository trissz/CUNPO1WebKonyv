ECMAScript szabványai

  1. ECMAScript 1 (1997): Az eredeti specifikáció, amely meghatározta a JavaScript nyelvet.
    Alapvető nyelvi konstrukciók, például változók, függvények, ciklusok stb.
  2. ECMAScript 2 (1998): Kisebb módosítások és javítások az ECMAScript 1-hez képest.
  3. ECMAScript 3 (1999): A legelterjedtebb verzió sok modern JavaScript kód még mindig támogatja.
    Új nyelvi elemek, például try/catch, reguláris kifejezések, kivételek, stb.
    Szigorú mód a javaslat környezeti biztonságának javítására.
  4. ECMAScript 4 (Elvetve fejlesztve, sosem adták ki): Nagyobb korszerűsítésekkel és funkciókkal tervezett verzió, de a fejlesztési folyamat során megszakadt.
  5. ECMAScript 5 (2009): Számos új nyelvi elem, például property getters/setters, JSON támogatás, szigorú módban folyó kód, stb.
    Kompatibilitás javítása régebbi böngészőkkel.
  6. ECMAScript 6 / ECMAScript 2015: Új funkciók és szintaktikai cukorkák, például osztályok, fat arrow függvények, modulrendszer, let és const változók stb.
    Generátorok és Promise objektumok bevezetése az aszinkron programozás támogatására.
  7. ECMAScript 2016 / ECMAScript 7: Kis méretű változtatások és kiegészítések, mint például a függvényargumentumokat default értékekkel vagy spread operátorral megadó szintaktikai lehetőség.
  8. ECMAScript 2017 / ECMAScript 8: Új funkciók, mint például az async/await szintaktikai cukorkák, Object.entries(), Object.values() stb.
  9. ECMAScript 2018 / ECMAScript 9: Új funkciók, mint például a Promise.finally(), Rest/spread properties, Aszinkron iterátorok stb.
  10. ECMAScript 2019 / ECMAScript 10: Új funkciók, mint például a Array.prototype.flat(), Array.prototype.flatMap(), Object.fromEntries() stb.
  11. ECMAScript 2020 / ECMAScript 11: Új funkciók, mint például a BigInt támogatás, globális import/export modulok stb.
  12. ECMAScript 2021 / ECMAScript 12: Új funkciók, mint például a Logical Assignment Operators, Promise.any(), WeakRef stb.
